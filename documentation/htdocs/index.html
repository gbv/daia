Document Availability Information (DAIA) Sourceforge project

The Document Availability Information Ontology (DAIA) describes the current availability of documents in libraries and similar institutions. Availability can be expressed in terms of specific services. See http://purl.org/NET/DAIA for a specification. This project collects implementation of DAIA in various programming languages so you can make use of it.

  • daia project page
  • Download files - stable releases from time to time
  • Browse the SVN-repository - always the current development
  • Get support

DAIA in other projects: VuFind contains a DAIA driver.
